# Markdown 文件範例

這是一個 Markdown 文件範例，展示了如何使用 Markdown 語法來撰寫文件內容。Markdown 是一種輕量級的標記語言，讓你可以使用簡單的符號來格式化文字。

## 標題

```markdown
# 這是 H1 標題

## 這是 H2 標題

### 這是 H3 標題

#### 這是 H4 標題

##### 這是 H5 標題

###### 這是 H6 標題
```

## 圖片

```markdown
![圖片說明](/vite.jpg)

![{small} 圖片說明](/vite.jpg)

![{medium} 圖片說明](/vite.jpg)

![{large} 圖片說明](/vite.jpg)
```
