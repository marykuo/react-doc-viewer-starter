# Setup

This documentation will guide you through the setup process for this React documentation viewer starter template. By following these steps, you'll be able to quickly create your own documentation website using Markdown and PDF files.

## Technology Stack

The project is built with React and Vite, which offers a fast development experience and optimized production builds. It also uses React Router for client-side routing, allowing you to create a multi-page documentation site.
